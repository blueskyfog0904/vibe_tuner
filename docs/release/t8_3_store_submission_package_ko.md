## 2) Play Store 문안
- 짧은 설명(80자 이내):
  `실시간 악기 튜너, 코드 검색/즐겨찾기, 메트로놈과 이어 트레이닝 지원`
- 상세 설명:
  `VibeTuner는 기타/우쿨렐레 연습에 필요한 핵심 도구를 하나로 제공합니다.`
  `실시간 튜너로 음정을 맞추고, 코드 라이브러리에서 폼을 빠르게 찾고, 메트로놈으로 박자를 안정화하세요.`
  `주요 기능:`
  `- 실시간 튜너(노이즈 게이트/감도/A4 기준 주파수 설정)`
  `- 코드 라이브러리(검색, 즐겨찾기, 최근 본 코드)`
  `- 메트로놈(탭 템포, 박자 설정)`
  `- 이어 트레이닝`

## 3) App Store 문안
- 부제(30자 내외): `튜너·코드·메트로놈`

- 키워드(100자 이내 예시):
